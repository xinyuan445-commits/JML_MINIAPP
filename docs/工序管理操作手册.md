# 工序管理系统 — 操作手册

**版本**：v1.0  
**适用范围**：管理后台（PC）+ 生产执行小程序（微信）

---

## 一、系统概述

工序管理系统由两个端组成：

| 端 | 使用人员 | 主要功能 |
|----|---------|---------|
| **管理后台**（PC 浏览器） | PMC / 管理员 | 配置工序、工站、工艺路线；给工单指定路线 |
| **生产小程序**（微信） | 车间操作工 | 扫工单开工、完工、工序间交接 |

---

## 二、整体流程

```
【管理员 - 后台配置】
  ①  定义工序     →   ②  定义工站     →   ③  建立工艺路线（工序顺序+工站）
                                                       ↓
【PMC - 工单指定】
  ④  在"工单路线指定"中，为每张待开工工单指定工艺路线
                                                       ↓
【操作工 - 小程序执行】
  ⑤  列表找到工单  →  ⑥  开工  →  ⑦  完工（填完工数量）
                                          ↓
                               ⑧  下一工序：接收 → 开工 → 完工
                                          ↓
                               ⑨  所有工序完成 → 工单状态变为"已完成"
```

---

## 三、管理后台操作说明

### 3.1 工序管理

**路径**：工单管理 → 工序管理

工序是生产流程中的最小操作单元，例如"裁切"、"压贴"、"包装"。

| 操作 | 步骤 |
|------|------|
| **新增工序** | 点击右上角「新增」→ 填写工序编码、工序名称 → 确认 |
| **编辑工序** | 点击列表中「编辑」按钮 → 修改后确认 |
| **启用 / 停用** | 点击「启用」或「停用」按钮切换状态 |

> ⚠️ 停用的工序不会出现在工艺路线配置中。

---

### 3.2 工站管理

**路径**：工单管理 → 工站管理

工站是实际执行工序的设备或位置，例如"压贴机1#"、"包装台A"。

| 操作 | 步骤 |
|------|------|
| **新增工站** | 点击右上角「新增」→ 填写工站编码、工站名称 → 确认 |
| **编辑工站** | 点击列表中「编辑」按钮 → 修改后确认 |
| **启用 / 停用** | 点击「启用」或「停用」按钮切换状态 |

---

### 3.3 工艺路线配置

**路径**：工单管理 → 工艺路线

工艺路线定义一类产品的生产流程：哪几道工序、按什么顺序、在哪个工站执行。

#### 新建路线流程

```
点击「新增路线」
    ↓
选择部门（以041开头的部门）
    → 路线编码自动生成（格式：RT-部门代码-流水号）
    → 路线名称默认填入部门名称（可修改）
    → 如果是该部门第一条路线，自动勾选"默认路线"
    ↓
填写完成后点击「保存」
    ↓
【自动弹出工序步骤配置抽屉 — 必须完成】
    → 点击「添加工序」
    → 选择工序 + 选择工站（必填）
    → 可拖动调整顺序
    → 点击「保存步骤」完成
```

> ⚠️ **注意**：新建路线后必须完成工序步骤配置，否则路线无法使用。若强制关闭抽屉，该路线将被自动删除。

#### 其他操作

| 操作 | 说明 |
|------|------|
| **编辑路线** | 点击「编辑」，可修改路线名称、默认设置等 |
| **配置工序步骤** | 点击「步骤」图标，可增删工序、调整顺序 |
| **删除路线** | 需要输入删除密码（管理员账号无需密码）|
| **设为默认** | 一个部门只能有一条默认路线；设置后，该部门新工单自动匹配此路线 |

---

### 3.4 工单路线指定

**路径**：工单管理 → 工单路线指定

将工艺路线与具体工单关联，工单才能在小程序中执行。

#### 页面说明

- **绿色标签**：该工单已通过部门默认路线自动匹配，无需手动操作
- **蓝色标签**：该工单已手动指定路线
- **灰色"未指定"**：该工单没有匹配的路线，需要手动指定

> 已开工或已完工的工单不会出现在此页面。

#### 单条指定

在「指定路线」列的下拉框中选择路线即可保存；清空下拉框即取消指定（取消后恢复显示默认路线或未指定）。

#### 批量指定

1. 勾选多条工单（支持全选）
2. 在顶部「批量选择路线」下拉框中选择路线
3. 点击「批量指定（N条）」按钮

## 四、生产小程序操作说明

小程序共有 **4 个界面**，按操作顺序依次说明。

---

### 4.1 现场管理首页（导航入口）

打开微信小程序后进入**现场管理**首页，页面以宫格卡片形式列出所有功能入口（例如：工序管理、其他现场功能等）。

**操作**：点击对应卡片即可进入该功能。

---

### 4.2 待开工列表页

**入口**：首页 → 工序管理

显示当前所有**待开工**工单（已指定路线、尚未开工）。

每张卡片显示：
- 工单号：行号（路线名称）
- 料号 / 产品名称
- 当前工序状态标签（待开工 / 进行中 / 待接收 / 已完成 / 无路线）
- 计划数量

**操作**：
- 下拉页面可刷新列表
- 点击任意工单卡片进入**明细页**

> 已开工或已完工的工单不在此列表中显示。

---

### 4.3 今日记录页（扫码 / 搜索）

**入口**：工序管理列表页顶部 → 扫码按钮 或 输入工单号搜索

此页面用于**快速定位工单**，同时展示当天的执行记录，分两个折叠区：

| 区域 | 内容 |
|------|------|
| **进行中** | 当天已开工、尚未完工的工序记录，显示工单号、工序名、开工时间 |
| **已完成** | 当天已完工的工序记录，显示工单号、工序名、开完工时间、完工数量 |

**两种查找方式**：

1. **手动输入**：在搜索栏输入工单号，点击键盘确认键搜索
2. **扫码**：点击右侧「扫码」按钮，扫描工单上的条码/二维码，自动跳转到对应工单的**明细页**

---

### 4.4 工单明细页

**入口**：从列表页或今日记录页点击工单进入

页面分为两个区域：

#### 上方：工单信息卡片

显示工单基本信息（工单号、产品名称、料号、生产部门、计划数量）及当前执行状态。

#### 中间：执行操作区（根据当前状态显示不同按钮）

| 当前状态 | 显示内容 | 可执行操作 |
|---------|---------|---------|
| **待开工** | 当前工序名称、可选工站 | 点击「**开工**」 |
| **进行中** | 当前工序名称、操作工站 | 点击「**完工**」→ 弹窗填写完工数量 |
| **待接收** | 上工序完工数量及备注 | 点击「**接收**」→ 弹窗填写接收数量 |
| **已完成** | 所有工序完成提示 | 可点击「**补工**」补录数量 |
| **无路线** | 提示联系 PMC 配置路线 | — |

#### 下方：子件清单

展示该工单所需物料的**应领数量**和**已领数量**，供操作工核对备料情况。

---

### 4.5 各操作详细说明

#### 开工

1. 在明细页确认当前工序名称
2. 若有多个工站可选，点击选择本次使用的工站（仅一个工站时自动显示，无需选择）
3. 点击「**开工**」

> 开工后其他人无法对同一工序重复开工。

#### 完工

1. 点击「**完工**」按钮
2. 弹窗中填写本次完工数量（参考值为子件应领数量）
3. 若填写数量低于参考值，需额外填写原因说明（必填）
4. 点击「**提交完工**」

完工后的流转：
- **非最后一道工序** → 下一工序进入"待接收"，等待下一工序操作工接收
- **最后一道工序** → 工单整体状态变为"已完成"

#### 接收（工序间交接）

1. 找到状态为"**待接收**"的工单，点击进入明细页
2. 查看上工序完工数量及备注说明
3. 点击「**接收**」，弹窗中填写本次接收数量
4. 若接收数量少于上工序完工数量，需填写差异原因（必填）
5. 点击「**确认接收**」→ 工单进入本工序"待开工"状态

#### 补工

若完工后发现数量有误，可在明细页底部找到「**补工**」入口（仅在有完工记录时显示），点击后填写补充数量提交。

## 五、常见问题

| 问题 | 原因 | 处理方式 |
|------|------|---------|
| 小程序中找不到工单 | 工单未指定工艺路线 | 在管理后台「工单路线指定」中处理 |
| 工单显示"无路线" | 工单部门没有匹配的默认路线，且未手动指定 | 在后台指定路线 |
| 无法开工，按钮不可点 | 上一道工序未完工/未交接 | 等待上一工序完成交接 |
| 工单在小程序中消失 | 工单已开工或完工 | 已正常流转，无需处理 |
| 无法删除工艺路线 | 密码错误 | 确认使用正确的删除密码；管理员账号无需密码 |
